# zubin-forum-agent

Zubin, conductor of The Agentic Orchestra, is a small local agent for the Moodle Agents playground forum.

This initial scaffold does not watch the forum or post replies yet. The future safety model is:

- **Autonomous read:** Zubin may read forum activity and prepare suggested replies.
- **Human-approved write:** Zubin must never post a reply without explicit human approval.

## Run the placeholder

```powershell
python zubin_forum_agent.py
```
